Introduction

Cutting board Designer helps you design end grain cutting boards.

It is inspired from Jayman7's excellent cbdesigner Windows program and provides much of the same functionality as a web based application, which allows its use on macs, chromebooks and tablets.

Tips

  • Drag wood strips to reorder them
  • Select multiple strips at one to change their size or wood type
